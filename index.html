What Juni AI Does

Juni AI is an AI-powered kids learning app for iPhone. Parents create the account, set up child profiles, and children learn through adaptive lessons across multiple subjects in a child-safe experience.

Subjects Available

  • Maths (Grades 1-6)
  • Reading (Grades 1-6)
  • Science (Grades 1-6)
  • Brain Puzzles (Grades 1-6)
  • World Explorer (Grades 2-6)
  • Coding Basics (Grades 3-6)
  • AI Basics (Grades 5-6)

Core Features

  • AI-generated adaptive lessons
  • Daily Challenge
  • Smart Recommendations
  • Mastery tracking by skill
  • Star rewards and streak system
  • Child Mode with PIN-locked parent access
  • Weekly email summaries for parents
  • Parent progress dashboard

Built for Parents

Parents are always the account holders. Children do not create accounts, cannot make purchases, and cannot access external links from child-facing screens.

Pricing

Free tier: 20 lessons per month, all subjects accessible, with a 7-day free trial on first signup.

Pro Monthly: $7.99/month

Pro Annual: $49.99/year

All subscriptions are handled through Apple In-App Purchase.
